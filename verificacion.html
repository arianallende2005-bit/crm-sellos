📋 Próximos Pasos para Ejecutar

  1. Instalar Node.js: Descargar e instalar Node.js 18+ desde nodejs.org
  2. Instalar PostgreSQL: Descargar PostgreSQL 15+ o usar Docker
  3. Backend - Instalar dependencias:
    cd backend
    npm install
  4. Backend - Inicializar BD:
    npm run init-db
  5. Backend - Iniciar servidor:
    npm run dev
  6. Frontend - Instalar dependencias (en otra terminal):
    cd frontend
    npm install
  7. Frontend - Iniciar app:
    npm start
  8. Abrir navegador: Ir a http://localhost:3000
  9. Login inicial: Usuario: admin / Password: cambiarme123

🐳 O usar Docker (Más Fácil)

  1. Instalar Docker Desktop para Windows
  2. Ejecutar: docker-compose up -d
  3. Acceder: http://localhost:3000
